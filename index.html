    Règles du Jeu

    • Le corps suit la tête du serpent.
    • Si le serpent mange un fruit, le corps grandit de 1 et on marque 10 points.
    • Si la tête touche le corps, c'est perdu.
    • Le plateau est un donut : le haut communique avec le bas, la gauche avec la droite.
    • Lorsque la partie est terminée, affichez le tableau des 20 meilleurs scores.
    • Si le joueur entre dans le top 20, un formulaire pour entrer le pseudo (minimum 6 caractères, maximum 20) s'affiche.
    • Le tableau des scores sera initialisé avec des données d'une API si vide.
    • Les scores sont conservés dans le stockage local.
